H.4. Getting The Source Via CVSup

The PostgreSQL source code is stored and managed using the CVS version control system.

At least three methods, anonymous CVS, rsync, and CVSup, are available to pull the CVS code tree from the PostgreSQL server to your local machine. Our Wiki, http://wiki.postgresql.org/index.php/Working_with_CVS, has additional details on working with CVS.
